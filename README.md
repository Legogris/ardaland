# Ardaland

Hosted on:
[https://ardaland.svt.se/](https://ardaland.svt.se/)

## Instructions

### Movement:
* Right/Left arrow keys (keyboard)
* Sliding the ship (touch screen)

### Shooting:
* Up arrow key (keyboard)
* Tapping on the screen (touch screen)
